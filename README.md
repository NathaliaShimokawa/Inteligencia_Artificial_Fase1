# Classificação de Câncer de Mama utilizando Machine Learning

## Descrição
Este projeto tem como objetivo desenvolver modelos de Machine Learning capazes de classificar tumores de mama como benignos ou malignos utilizando o Breast Cancer Wisconsin Dataset. Também foram aplicadas técnicas de Inteligência Artificial Explicável para interpretar as decisões do modelo.

## Tecnologias utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- SHAP
- KaggleHub

## Como executar
git clone https://github.com/NathaliaShimokawa/Inteligencia_Artificial_Fase1.git

cd Inteligencia_Artificial_Fase1

pip install -r requirements.txt

jupyter notebook

## Dataset
Dataset utilizado: https://www.kaggle.com/datasets/uciml/breast-cancer-wisconsin-data
O dataset é baixado automaticamente durante a execução do notebook por meio da biblioteca KaggleHub
